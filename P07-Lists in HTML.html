Lists are of 2 types: 1. Unordered List 2. Ordered List

Lists in HTML

Unordered List

  • List of foods...

  • Bread
  • Butter
  • Jam

Ordered List

  1. List of foods...

  2. Bread
  3. Butter
  4. Jam

We can also define the type of ordered lists (e.g., 1-2-3, A-B-C, a-b-c)

    List with type A-B-C

  1. Bread
  2. Butter
  3. Jam

    List with type a-b-c

  1. Bread
  2. Butter
  3. Jam

    List with type 1-2-3 (by default and can also be changed)

  1. Bread
  2. Butter
  3. Jam

    List with type I-II-III

  1. Bread
  2. Butter
  3. Jam

    List with type i-ii-iii

  1. Bread
  2. Butter
  3. Jam
