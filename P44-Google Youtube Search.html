We can directly open Google, YouTube with search results from our form submission using the 'name' attribute

    1. Open YouTube in the browser
    2. Search for something (e.g., 'Apple')
    3. After clicking the search button in YouTube, observe the URL:
    4. https://www.youtube.com/results?search_query=Apple
    5. The action used is: https://www.youtube.com/results
    6. The name used is: search_query
    7. Creating a form with type="text," the same name, and action will append the input to the URL after submission.
    8. It will redirect to YouTube with the completed URL.
