"Simple Input": A basic input field with no specified type. Useful for generic text input.

"Text Input": Accepts alphanumeric characters. Suitable for short text entries.

"Password Input": Conceals entered text for sensitive information like passwords.

"Number Input": Allows entry of numeric values, restricting non-numeric input.

"Time Input": Permits the selection of a time. Useful for time-related input.

"Color Input": Enables the selection of a color using a color picker.

"Date Input": Allows users to pick a date from a calendar.

"Email Input": Validates input as an email address, assisting in correct formatting.

"File Input": Allows users to upload files to the server.

"Checkbox": Presents a checkbox for binary (true/false) choices.

"Radio Button": Provides a set of mutually exclusive options. Only one can be selected.

"Range Input": Allows users to select a numeric value within a specified range.

"Search Input": Provides a text input specifically designed for search queries.

"URL Input": Validates input as a URL.

"Tel Input": Accepts telephone numbers.

"Submit Button": Submits the form when clicked.

"Reset Button": Resets form fields to their default values.

"Image Button": Similar to a submit button but displays an image.
