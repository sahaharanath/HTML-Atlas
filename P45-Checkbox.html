Let's learn about Checkbox:

(1) All checkboxes are by default unchecked


(2) To make checkboxes by default checked, 'checked' attribute is used inside input

When the checkbox is checked and the form is submitted, the URL will include the parameter 'age=on' (where n is a value assigned to the checkbox).

If the checkbox is not checked, the URL won't include the 'age' parameter.

This mechanism is commonly used to pass boolean values through URLs, such as for filtering or conditional actions.
