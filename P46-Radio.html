Understanding Radio Elements:

Only one option can be selected at a time. Note: When one option is selected, any previously selected option will be unselected, as all options are connected through the name attribute inside the input element.


1st Case: Choose your favorite subject


Here, if we select any option and submit, notice the URL: http://localhost:63342/action?Subjects=on. In this case, no data is passed through the URL. To pass data through the URL using the radio element, we use the "value" attribute. As we've seen before, the 'name' attribute passes data through the URL, and here 'name' keeps all options connected. We will use the 'value' attribute to pass data through the URL.
