Let's learn about Block Element v/s Inline Element

  • Block Element : A block-level element is an HTML element that begins on a new line and extends the full width of the parent container. It creates a "block" of content.

  • Inline Element : An inline-level element is an HTML element that does not start on a new line and only takes up as much width as necessary. It flows within the content and does not create a new "block."


Heading Tag & Paragraph Tag are examples of block elements.

Anchor Tag & Image Tag are examples of block elements.
