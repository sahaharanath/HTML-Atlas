We have 3 types of button attributes! Those are:

  • type="submit" → Used to submit data in a form.
  • type="button" → Used to trigger a custom JavaScript action.
  • type="reset" → Used to reset all filled data in a form.

Login Form

Additional Information:

  • Submit Button: Submits the form data to the server.
  • Reset Button: Resets all filled form fields to their default values.
  • Custom Button: Can be customized with JavaScript for specific actions.
