Let's learn about 'Div Element'

It is a block element.


Here we can see that all the references are in the same line :

Now if we want to keep 'Google,' 'Facebook,' 'Instagram' in one line and 'Twitter' in another line, we will take help of 'Div'...

Twitter

Here, as the 'div' is a block element, it is occupying the whole width and keeping all the elements inside it that are given.
